# Examples

This folder contains example project outputs for the film pipeline workflow.

## Included Example

- `internal-mg-demo`
  - A compact internal motion-graphics sample project
  - Shows the expected file layout from Step 1 through Step 5
  - Demonstrates the MG Step 3 alias:
    - `03_step3_motion_design_package.md`

## Notes
