# Step 3 - Motion Design Package

## Design System

- Typography: bold sans for headlines, neutral sans for detail
- Layout: 16:9 safe center, adaptable to 9:16 crop
- Palette:
  - Deep blue `#164B8B`
  - Teal `#16B3A3`
  - White `#F8FAFC`
  - Dark gray `#24313F`
  - Light gray `#D9E1EA`

## Graphic Elements

- rounded UI cards
- message bubbles
- checklist marks
- arrows and connector lines
- tabs and status chips
- split-screen compare blocks

## Motion Rules

- average shot transition: 6-10 frames
- text should enter from one direction only per scene
- use scale + fade for emphasis, not flashy effects
- avoid 3D camera moves
- keep each key message readable within 1 second

## Copy Rules

- one message per screen
- max headline length: 10 Chinese characters if possible
- verbs should be practical: 整理 / 总结 / 输出 / 使用
- avoid buzzwords like 智能革命 / 未来办公 / 颠覆

## Asset List

- logo or internal tool mark
- three functional icons
- one simplified UI panel mockup
- one internal access badge or platform marker

## Editing Notes

- prioritize subtitle readability over decorative motion
- if needed, compress Shot 3 and Shot 4 before cutting CTA
- this version should feel like enterprise enablement, not advertising spectacle
