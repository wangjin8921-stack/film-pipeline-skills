# Internal MG Demo

This example shows a compact internal motion-graphics project produced through the film pipeline.

## Files

- `00_project_summary.md`
- `01_step1_directing_brief.md`
- `02_step2_storyboard.md`
- `03_step3_motion_design_package.md`
- `04_step4_execution_plan.md`
- `05_step5_generation_prompts.md`

## Why This Example Matters

- It demonstrates the expected output shape for a full Step 1 to Step 5 run.
- It uses the MG-specific Step 3 filename, which is supported by the plugin resume logic.
- It shows a practical internal-promo workflow rather than a cinematic live-action case.

## Companion Output Note

In newer repository versions, Step 5 may also generate an optional Chinese companion file:

- `05_step5_generation_prompts_zh.md`

That companion file is not required for completion tracking, but it can be useful for review and Chinese prompt experimentation.
