# BV AIGC Style Case

Use this file as a visual-style calibration case, not as a structure template.

This reference captures one short AIGC-driven visual promo pattern from the local benchmark video `BV.mp4`.

## Why This Case Matters

This case is useful because it is:

- clearly style-led rather than information-led
- not a valid `MG Explainer / Information Short` benchmark
- a strong example of poster-motion, title-promo, and teaser-style AIGC output
- a good reminder that some references should shape surface language only, not step structure

Use it when the user wants:

- a short AIGC visual teaser benchmark
- a poster-like motion reference
- an AI-generated style reference with strong atmosphere
- a title-led or key-art-led promo feeling

Do not use it to define storyline, explainer structure, or MG execution logic.

## Core Case Summary

Working description:

- a short-form AIGC visual promo
- Chinese character-led title treatment
- painterly snow-mountain and Tibetan-inspired environment
- one central heroine-like figure with flowing costume and strong poster composition
- text treatment such as `AI辅助生成`, `MAKE A MOVIE BY OF AIGC`, and date/title overlays

Approximate role:

1. establish visual motif
2. present heroine and atmosphere
3. intensify title and key-art composition
4. close as a teaser or motion-poster statement

## Dominant Type Judgment

Primary type:

- `Trailer / Teaser Style Short`

Secondary supporting traits:

- `Concept Narrative Short`
- `Typography-Led Promo`
- `Poster-Motion / Key-Art Animation`

Not primary type:

- `MG Explainer / Information Short`
- `UI Demo / Screenflow Video`
- `Data Visualization Film`
- `Workflow-led enterprise explainer`

## Why It Is Not MG

This case is not a good MG benchmark because:

- it does not explain a workflow, product, or information hierarchy
- it is driven by visual impact rather than conceptual clarity
- the text acts more like title treatment than structured communication
- the piece behaves like a motion poster or teaser
- there is no modular explainer logic to transfer into Step 2 through Step 4 MG planning

## What This Case Is Good For

Transferable style signals in this case:

- atmosphere-first framing
- strong hero-image composition
- painterly environment treatment
- title and date overlays as part of the image
- teaser-level compression
- AI-generated beauty and spectacle as the main appeal

## What Is Specific To This Case

These are case-specific choices, not universal visual-style rules:

- Tibetan or snow-domain visual language
- white, red, black, and icy blue palette
- long-braid heroine styling
- ink-wash and painterly mountain treatment
- calligraphic Chinese title treatment
- date-stamped event or release framing

Do not generalize these details to every visual-style promo.

## What Must Not Be Overgeneralized

Do not conclude from this case that:

- all AIGC visual promos need Chinese fantasy styling
- all teaser-style shorts need a heroine portrait
- all visual-style references should use ink-wash mountains
- all title promos should use poster composition only
- all short-form style references can substitute for structure references

## Reusable Guardrails

These lessons transfer well to future style-led AIGC promos:

- prioritize one dominant visual motif
- keep the shot count small and the image impact high
- let typography and imagery reinforce one another
- treat the piece like key art in motion
- compress meaning into image, title, and atmosphere rather than explanation

## Reusable Freshness Levers

These are safe places to vary future visual-style promos:

- world or cultural motif
- palette system
- title treatment
- key-art composition
- texture style
- transition energy
- hero styling
- motion-poster framing

## Step 1 Calibration

What this case teaches Step 1:

- a reference can influence `visual direction` without determining `video_type`
- teaser and key-art style references belong to the surface language layer
- the core communication goal must still determine the project type

What Step 1 should not do:

- relabel an explainer as a teaser just because the user likes this visual style
- mistake a visual-style benchmark for a structural benchmark

## Step 2 To Step 4 Calibration

What this case can influence:

- motif selection
- palette direction
- title treatment
- poster-like hero shot design
- teaser pacing and reveal energy

What this case should not directly dictate:

- explainer section logic
- prompt modularity for MG
- execution batching for information-led videos
- UI, chart, or workflow-based asset planning

## Recommended Use Rule

If a future project references this case:

- use it as a visual-style benchmark
- transfer only motif, atmosphere, title, and poster-motion lessons
- keep structural logic tied to the real project type

If the project is actually an explainer, workflow film, or product education piece:

- do not let this case replace the correct structural benchmark
- combine it with a type-appropriate benchmark only if the user explicitly wants a hybrid
