# Video Type Profiles

Use this file when `video_type` needs to shape the Step 1 brief.

For each project, pick one dominant lane first. Only blend types when the user explicitly asks for a hybrid.

## How To Use These Profiles

Treat each profile as two layers:

- `Guardrails`: the communication goal, structure, rhythm, visual bias, and avoid-lines already listed under the profile
- `Freshness Levers`: the parts that can still change without breaking the type

When writing Step 1:

- lock the guardrails first
- choose 2-4 freshness levers on purpose
- make the freshness specific enough that the concept does not feel templated
- do not use freshness as an excuse to cross into the wrong format

## Freshness Levers By Type Family

These are not replacements for the per-type profiles below. Use them as safe variation zones inside the chosen lane.

### Information-Led Types

Examples:

- `MG Explainer / Information Short`
- `Data Visualization Film`
- `UI Demo / Screenflow Video`
- `Training / Process Explainer Short`
- `Policy / Compliance Communication`

Good freshness levers:

- information metaphor
- module or card system
- icon language
- transition grammar
- comparison structure
- title hierarchy style

Keep locked:

- clarity first
- readable sequence
- low ambiguity
- low dependency on character acting

### People-Led Types

Examples:

- `Interview Film`
- `Customer Testimonial`
- `Leadership Message Film`
- `Recruitment Promo`
- `Street Interview / Vox Pop`

Good freshness levers:

- location character
- support-cutaway motif
- quote chaptering logic
- subtle recurring visual symbol
- evidence style
- emotional progression framing

Keep locked:

- credibility
- speaker or protagonist continuity
- understandable message flow
- restrained motion when trust matters

### Hype-Led Types

Examples:

- `Warm-Up / Hype Video`
- `Opening Video`
- `Event Teaser`
- `Flash / Quick-Cut Montage`
- `Trailer / Teaser Style Short`

Good freshness levers:

- rhythm pattern
- slogan architecture
- reveal mechanic
- graphic hit system
- sound-reactive visual logic
- recurring impact motif

Keep locked:

- momentum
- strong readability
- short idea units
- no long explanatory detours

### Trust-Led Types

Examples:

- `Enterprise Promo`
- `Brand Image Film`
- `Documentary-Style Enterprise Film`
- `Case Story Short`

Good freshness levers:

- proof angle
- motif choice
- progression spine
- environmental emphasis
- before-and-after framing
- symbolic payoff image

Keep locked:

- credibility
- coherent tone
- proof readability
- no hype-style over-fragmentation

### Narrative-Led Types

Examples:

- `Concept Narrative Short`
- `Emotional Atmosphere Short`

Good freshness levers:

- central metaphor
- transformation mechanism
- recurring object or place
- color-emotion progression
- scene reveal pattern
- emotional punctuation

Keep locked:

- one clear thematic spine
- coherent transformation arc
- manageable continuity
- no random idea pileup

## Product Promo Short

- Main goal: sell a product value or feature set
- Best for: product launches, feature awareness, conversion-minded short ads
- Structure: hook -> pain point -> solution -> payoff -> CTA
- Rhythm: fast and clear
- Visual bias: benefit-first, demo-friendly, feature-led
- Avoid: long brand narrative before the product appears

## Enterprise Promo

- Main goal: communicate company image, capability, trust, and positioning
- Best for: official brand communication, external company overview
- Structure: who we are -> what we do -> why it matters -> why trust us
- Rhythm: steady, credible, polished
- Visual bias: brand tone, environment, people, scale, achievement
- Avoid: pure feature dumping or hard-sell ad language

## Brand Image Film

- Main goal: shape brand feeling more than explain features
- Best for: premium branding, positioning refresh, flagship communication
- Structure: proposition -> emotional proof -> brand world -> final statement
- Rhythm: controlled and elevated
- Visual bias: mood, design language, signature motifs, premium pacing
- Avoid: dense operational detail

## Internal Promo / Internal Adoption Short

- Main goal: help employees understand and adopt something quickly
- Best for: internal tools, policy rollout, workflow change, training kickoff
- Structure: daily problem -> practical use -> how to start -> immediate benefit
- Rhythm: direct and practical
- Visual bias: clarity, utility, familiarity, low jargon
- Avoid: grand brand messaging or overhyped future-tech framing

## Opening Video

- Main goal: create atmosphere and frame the event or meeting
- Best for: conferences, annual meetings, launch sessions, leadership entrances
- Structure: energy build -> thematic statement -> reveal -> stage handoff
- Rhythm: high control, often music-led
- Visual bias: typography, slogan, scale, emotional lift
- Avoid: dense instruction, detailed process explanation, feature tutorials

## Warm-Up / Hype Video

- Main goal: raise room energy before a live moment
- Best for: annual meetings, gala nights, sales kickoffs, stage countdowns
- Structure: build -> intensify -> peak -> handoff
- Rhythm: punchy and music-driven
- Visual bias: impact, rhythm cuts, slogans, crowd energy, momentum
- Avoid: slow exposition or training-style information density

## Event Teaser

- Main goal: build interest before an event
- Best for: registration pushes, internal event warm-up, summit promo
- Structure: what is coming -> why it matters -> key cue -> CTA
- Rhythm: sharp and anticipatory
- Visual bias: countdown, motion typography, agenda hints, reveal beats
- Avoid: fully explaining the whole event upfront

## Event Recap

- Main goal: compress highlights into a strong memory
- Best for: event summary, internal reporting, social recap
- Structure: arrival -> peak moments -> reactions -> summary takeaway
- Rhythm: highlight-driven
- Visual bias: momentum, milestone moments, participation, applause points
- Avoid: chronological over-detail

## Same-Day Edit / Flash Recap

- Main goal: create immediate event heat from fresh footage
- Best for: conferences, brand activations, roadshows, gala nights
- Structure: quick highlights -> strongest moments -> emotional payoff
- Rhythm: very fast
- Visual bias: crowd energy, stage beats, applause, fast montage
- Avoid: heavy narrative complexity or lots of explanatory captions

## MC / Guest Intro Video

- Main goal: set up a speaker, performer, host, or guest entrance
- Best for: gala programs, forums, award nights, internal ceremonies
- Structure: identity -> credibility or hook -> dramatic lead-in
- Rhythm: compact and ceremonial
- Visual bias: name, title, highlights, event tone
- Avoid: trying to tell the whole life story

## Program Interstitial / Segment Bumper

- Main goal: bridge one live segment to the next cleanly
- Best for: gala nights, award shows, internal conferences
- Structure: title -> visual cue -> transition out
- Rhythm: short and functional
- Visual bias: motion branding, title cards, segment identity
- Avoid: unnecessary narrative expansion

## Interview Film

- Main goal: let a speaker deliver a clear point with authenticity
- Best for: founder interviews, internal leadership interviews, employee stories
- Structure: framing question -> core statements -> takeaway
- Rhythm: medium and readable
- Visual bias: speaker credibility, reaction pauses, clean cutaways
- Avoid: too many unrelated topics in one short piece

## Customer Testimonial

- Main goal: build trust through first-person credibility
- Best for: customer proof, case validation, B2B persuasion
- Structure: challenge -> use -> result -> endorsement
- Rhythm: moderate, proof-led
- Visual bias: believable environment, honest tone, clear outcomes
- Avoid: generic praise with no specifics

## Case Story Short

- Main goal: show a believable before-and-after transformation
- Best for: customer stories, employee success stories, applied use cases
- Structure: situation -> problem -> intervention -> improved result
- Rhythm: medium pace, proof-oriented
- Visual bias: clarity, credibility, concrete use context
- Avoid: abstract storytelling with no real outcome

## Street Interview / Vox Pop

- Main goal: gather quick reactions or opinions
- Best for: social clips, campaign reactions, awareness checks
- Structure: setup question -> answer fragments -> summary beat
- Rhythm: quick and varied
- Visual bias: spontaneity, real voices, contrast between responses
- Avoid: overproducing it until it loses authenticity

## Training / Process Explainer Short

- Main goal: teach a process with low ambiguity
- Best for: SOP intros, quick-start videos, internal operations education
- Structure: what this is -> steps -> result -> reminder
- Rhythm: structured and instructional
- Visual bias: sequence, labels, process markers, readable hierarchy
- Avoid: decorative concepting that hides the process

## Policy / Compliance Communication

- Main goal: make a rule or policy understandable and actionable
- Best for: internal policy rollout, compliance reminders, safety communication
- Structure: what changed -> what matters -> what to do -> reminder
- Rhythm: direct and sober
- Visual bias: clarity, emphasis, hierarchy, low ambiguity
- Avoid: vague slogans without behavioral instruction

## Recruitment Promo

- Main goal: attract the right talent and shape employer image
- Best for: hiring campaigns, team introductions, culture recruitment
- Structure: role/context -> team value -> growth or mission -> invitation
- Rhythm: energetic but human
- Visual bias: people, environment, work meaning, team energy
- Avoid: empty slogans with no sense of role or culture

## Leadership Message Film

- Main goal: package a leadership message with authority and warmth
- Best for: annual updates, strategic announcements, internal morale communication
- Structure: context -> message -> direction -> call to action
- Rhythm: calm and credible
- Visual bias: authority, trust, focus, internal alignment
- Avoid: over-stylizing the message until it feels distant

## MG Explainer / Information Short

- Main goal: explain quickly through graphic logic
- Best for: tools, process explanation, platform education, internal communication
- Structure: problem -> simplification -> steps -> result
- Rhythm: brisk, readable, modular
- Visual bias: icons, cards, charts, UI blocks, text-led screens
- Avoid: fake cinematic realism or unnecessary character acting

## Data Visualization Film

- Main goal: make numbers legible and persuasive
- Best for: annual reporting, sales dashboards, business updates, investor communication
- Structure: metric context -> key number -> implication -> summary
- Rhythm: steady and information-led
- Visual bias: charts, counters, infographics, guided emphasis
- Avoid: flooding the screen with too many numbers at once

## UI Demo / Screenflow Video

- Main goal: show how a product or interface works
- Best for: software launches, internal tool onboarding, feature adoption
- Structure: task -> interaction -> output -> value
- Rhythm: practical and screen-readable
- Visual bias: cursor, UI states, zoom emphasis, clean callouts
- Avoid: abstract promises without showing the interface

## Flash / Quick-Cut Montage

- Main goal: create momentum and attention through speed
- Best for: campaign bursts, youth-facing edits, event hype, social cutdowns
- Structure: rapid fragments -> pattern -> payoff
- Rhythm: very fast
- Visual bias: short beats, typography hits, fast transitions, music sync
- Avoid: complex explanation that needs long reading time

## Concept Narrative Short

- Main goal: land a strong idea through a simple story
- Best for: concept-led promos, emotional thematic pieces, visual essays
- Structure: hook -> progression -> transformation -> payoff
- Rhythm: controlled and expressive
- Visual bias: metaphor, motif, continuity, emotional arc
- Avoid: overloading it with too many facts or product bullets

## Emotional Atmosphere Short

- Main goal: make the viewer feel before they fully analyze
- Best for: culture, mission, tribute, human-centered messaging
- Structure: emotional setup -> resonance -> uplift or reflection
- Rhythm: slower and more breathing-room oriented
- Visual bias: mood, texture, silence, gesture, emotional pacing
- Avoid: cluttered exposition or heavy feature explanation

## Trailer / Teaser Style Short

- Main goal: generate anticipation
- Best for: campaigns, launches, upcoming events, dramatic reveals
- Structure: fragments -> escalation -> withheld reveal -> impact ending
- Rhythm: punchy, tension-driven
- Visual bias: contrast, momentum, curiosity, reveal timing
- Avoid: over-explaining too early

## Documentary-Style Enterprise Film

- Main goal: feel real, grounded, and observational
- Best for: manufacturing, ESG, human stories, company culture, mission stories
- Structure: situation -> observation -> human detail -> meaning
- Rhythm: measured and authentic
- Visual bias: real spaces, real people, natural continuity
- Avoid: ad-copy voice dressed up as documentary

## Type Selection Rules

- `产品宣传片`, `产品广告`, `产品短片` -> `Product Promo Short`
- `企业宣传片`, `企业介绍片` -> `Enterprise Promo`
- `品牌片`, `品牌形象片` -> `Brand Image Film`
- `内部推广片`, `内部宣传片`, `内部启用片` -> `Internal Promo / Internal Adoption Short`
- `开场视频`, `开场片`, `大会开场` -> `Opening Video`
- `暖场视频`, `热场视频`, `hype video` -> `Warm-Up / Hype Video`
- `活动预热片`, `预热视频` -> `Event Teaser`
- `活动回顾片`, `花絮回顾`, `精彩回顾` -> `Event Recap`
- `快闪回顾`, `当天快剪`, `same-day edit` -> `Same-Day Edit / Flash Recap`
- `嘉宾介绍片`, `主持人出场片`, `领导出场片` -> `MC / Guest Intro Video`
- `串场片`, `转场片`, `节目包装片` -> `Program Interstitial / Segment Bumper`
- `采访片`, `人物采访片`, `访谈片` -> `Interview Film`
- `客户证言片`, `客户证言`, `testimonial` -> `Customer Testimonial`
- `案例片`, `案例故事片` -> `Case Story Short`
- `街访`, `vox pop` -> `Street Interview / Vox Pop`
- `培训片`, `教程片`, `操作说明片`, `流程讲解片` -> `Training / Process Explainer Short`
- `制度宣导片`, `合规宣导片`, `安全宣导片` -> `Policy / Compliance Communication`
- `招聘片`, `雇主品牌片` -> `Recruitment Promo`
- `领导致辞片`, `领导讲话包装片` -> `Leadership Message Film`
- `MG`, `MG动画`, `信息动画`, `纯图形动画` -> `MG Explainer / Information Short`
- `数据片`, `数据可视化片`, `年报数据视频` -> `Data Visualization Film`
- `UI演示`, `屏幕演示`, `产品演示视频` -> `UI Demo / Screenflow Video`
- `快闪片`, `快切混剪`, `节奏混剪` -> `Flash / Quick-Cut Montage`
- `概念短片`, `概念片` -> `Concept Narrative Short`
- `情绪短片`, `氛围短片`, `情感短片` -> `Emotional Atmosphere Short`
- `预告片`, `teaser`, `trailer` -> `Trailer / Teaser Style Short`
- `纪录片风格企业片`, `纪实企业片` -> `Documentary-Style Enterprise Film`

If the label and the actual goal conflict, choose the profile that best matches the user's real communication goal, not just the surface name.
