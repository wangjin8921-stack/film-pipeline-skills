# Storyboard Type Profiles

Use this file to convert a Step 1 `video_type` into the right storyboard behavior.

This file is about shot planning, not high-level positioning.

## How To Use These Profiles

Treat the listed focus, section logic, shot bias, count tendency, and avoid-lines as storyboard guardrails.

Then choose a few freshness levers that can make the storyboard feel specific without breaking the format.

Use freshness in areas such as:

- recurring shot motif
- section entry pattern
- insert logic
- comparison or reveal structure
- how support shots reinforce the core message

Do not use freshness to ignore the type's shot logic.

## Storyboard Freshness Levers By Type Family

### Information-Led Types

Good freshness levers:

- card reveal rhythm
- compare-state structure
- icon-to-real-object handoff
- repeated process marker
- chapter label system

Keep locked:

- step order
- reading clarity
- low ambiguity
- minimal decorative detours

### People-Led Types

Good freshness levers:

- support-cutaway motif
- chaptering by quote theme
- environmental proof style
- gesture detail inserts
- visual evidence progression

Keep locked:

- speaker continuity
- quote readability
- support coverage discipline
- trust-first pacing

### Hype-Led Types

Good freshness levers:

- reveal cadence
- rhythm family
- title or slogan strike pattern
- crowd or scale punctuation
- repetition mechanic

Keep locked:

- acceleration
- short beat design
- quick reading time
- minimal exposition

### Trust-Led Types

Good freshness levers:

- proof order
- symbolic establishing shots
- process-detail motif
- outcome framing
- credibility reveal pattern

Keep locked:

- coherent tone
- believable environments
- proof before flourish
- low gimmick density

### Narrative-Led Types

Good freshness levers:

- motif recurrence
- transformation milestones
- recurring object logic
- emotional contrast placement
- reveal sequence

Keep locked:

- one clear story spine
- manageable shot continuity
- controlled escalation
- payoff clarity

## Product Promo Short

- Storyboard focus: product clarity and payoff speed
- Section logic: hook -> problem -> product intervention -> outcome -> CTA
- Shot bias: feature moments, reaction moments, result moments
- Shot count tendency: medium
- Avoid: hiding the product until too late

## Enterprise Promo

- Storyboard focus: credibility and company scale
- Section logic: identity -> capability -> proof -> trust close
- Shot bias: environment, team, process, result, scale indicators
- Shot count tendency: medium
- Avoid: ad-like hard selling

## Brand Image Film

- Storyboard focus: atmosphere and identity
- Section logic: mood -> symbolic proof -> brand world -> statement
- Shot bias: hero visuals, motif shots, elevated transitions
- Shot count tendency: low to medium
- Avoid: over-explaining operations

## Internal Promo / Internal Adoption Short

- Storyboard focus: practical understanding
- Section logic: daily pain -> simple use -> immediate benefit -> start now
- Shot bias: familiar workflow moments, direct use logic, visible outcome
- Shot count tendency: medium
- Avoid: cinematic detours that reduce clarity

## Opening Video

- Storyboard focus: impact and stage handoff
- Section logic: build -> reveal -> thematic peak -> handoff
- Shot bias: large gestures, slogan moments, reveal shots
- Shot count tendency: low to medium
- Avoid: too many explanatory shots

## Warm-Up / Hype Video

- Storyboard focus: energy escalation
- Section logic: pulse -> intensify -> peak
- Shot bias: rhythm hits, crowd or stage cues, impact typography
- Shot count tendency: high
- Avoid: informational scenes that slow tempo

## Event Teaser

- Storyboard focus: anticipation
- Section logic: signal -> promise -> hint -> CTA
- Shot bias: reveal fragments, countdowns, agenda hints
- Shot count tendency: low to medium
- Avoid: fully resolving the mystery too early

## Event Recap

- Storyboard focus: compress highlights
- Section logic: arrival -> highlight stack -> emotional close
- Shot bias: moments, faces, applause, milestones
- Shot count tendency: medium to high
- Avoid: flat chronological logging

## Same-Day Edit / Flash Recap

- Storyboard focus: immediacy
- Section logic: strongest moments only
- Shot bias: fast peaks, crowd moments, logos, stage highlights
- Shot count tendency: high
- Avoid: narrative complexity or too much text

## MC / Guest Intro Video

- Storyboard focus: entrance framing
- Section logic: identity -> credibility -> impact reveal
- Shot bias: name cards, title moments, prestige cues
- Shot count tendency: low
- Avoid: too much biography

## Program Interstitial / Segment Bumper

- Storyboard focus: clean transition
- Section logic: title -> cue -> out
- Shot bias: branded motion cards, segment markers
- Shot count tendency: very low
- Avoid: expanding into a mini-story

## Interview Film

- Storyboard focus: quote support
- Section logic: speaker setup -> idea blocks -> takeaway
- Shot bias: main interview angle, secondary angle, b-roll evidence
- Shot count tendency: low to medium
- Avoid: random visual variety that breaks speaker continuity

## Customer Testimonial

- Storyboard focus: trust and specificity
- Section logic: problem -> solution use -> measurable result
- Shot bias: speaker proof, environment proof, outcome proof
- Shot count tendency: low to medium
- Avoid: empty praise without visual evidence

## Case Story Short

- Storyboard focus: transformation
- Section logic: before -> friction -> intervention -> after
- Shot bias: contrast shots, process beats, result shots
- Shot count tendency: medium
- Avoid: vague "success" with no before/after structure

## Street Interview / Vox Pop

- Storyboard focus: answer variation
- Section logic: question -> rapid response blocks -> synthesis
- Shot bias: answer fragments, reaction inserts, summary text
- Shot count tendency: high
- Avoid: long monologues from one person

## Training / Process Explainer Short

- Storyboard focus: step clarity
- Section logic: setup -> step 1 -> step 2 -> step 3 -> result
- Shot bias: instruction shots, labels, process markers
- Shot count tendency: medium
- Avoid: too much decorative coverage

## Policy / Compliance Communication

- Storyboard focus: behavior clarity
- Section logic: what changed -> what it means -> what to do
- Shot bias: clear scenario examples, rule markers, consequences or reminders
- Shot count tendency: low to medium
- Avoid: ambiguous metaphor-heavy storytelling

## Recruitment Promo

- Storyboard focus: role appeal and team feel
- Section logic: environment -> people -> growth/value -> invitation
- Shot bias: team moments, role moments, atmosphere, mission
- Shot count tendency: medium
- Avoid: generic culture slogans unsupported by visuals

## Leadership Message Film

- Storyboard focus: authority and direction
- Section logic: context -> message -> call to action
- Shot bias: leader anchor shot, support cutaways, key text lines
- Shot count tendency: low
- Avoid: excessive stylistic fragmentation

## MG Explainer / Information Short

- Storyboard focus: information flow
- Section logic: problem -> simplify -> steps -> result
- Shot bias: cards, icons, labels, UI blocks, compare screens
- Shot count tendency: medium
- Avoid: writing shots like live-action scenes

## Data Visualization Film

- Storyboard focus: metric readability
- Section logic: context -> number -> implication -> next number
- Shot bias: chart reveals, counters, trend comparisons
- Shot count tendency: medium
- Avoid: too many numbers per shot

## UI Demo / Screenflow Video

- Storyboard focus: interface understanding
- Section logic: task -> click path -> output -> value
- Shot bias: UI states, zoom-ins, callouts, cursor moments
- Shot count tendency: medium
- Avoid: abstract talking points without showing the screen

## Flash / Quick-Cut Montage

- Storyboard focus: speed and emphasis
- Section logic: hit -> hit -> hit -> payoff
- Shot bias: very short visual beats, text bursts, sync moments
- Shot count tendency: high
- Avoid: long explanatory frames

## Concept Narrative Short

- Storyboard focus: metaphor progression
- Section logic: setup -> escalation -> transformation -> payoff
- Shot bias: visual motif continuity, symbolic hero shots
- Shot count tendency: medium
- Avoid: random scene hopping with no thematic spine

## Emotional Atmosphere Short

- Storyboard focus: feeling progression
- Section logic: mood -> resonance -> uplift or reflection
- Shot bias: slower emotional beats, detail shots, breathing room
- Shot count tendency: low to medium
- Avoid: dense copy overlays

## Trailer / Teaser Style Short

- Storyboard focus: anticipation and reveal control
- Section logic: fragments -> acceleration -> impact close
- Shot bias: withheld information, dramatic beats, reveal frames
- Shot count tendency: medium to high
- Avoid: explaining the whole narrative

## Documentary-Style Enterprise Film

- Storyboard focus: lived-in authenticity
- Section logic: real-world setup -> human/process detail -> meaning
- Shot bias: observational coverage, real spaces, real gestures
- Shot count tendency: medium
- Avoid: over-stylized ad coverage that breaks authenticity

## Type Selection Rules

- If Step 1 already chose a profile, follow it.
- If the user asks Step 2 to change the format, honor the user's new format and note the shift.
- If a type suggests fewer shots, do not force ad-style fragmentation.
- If a type suggests more rhythm, shorten shot purposes and simplify continuity demands.
