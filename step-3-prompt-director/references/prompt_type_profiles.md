# Prompt Type Profiles

Use this file to adapt Step 3 output to the actual film format.

This file is about prompt behavior, execution mode bias, and batch strategy.

## How To Use These Profiles

Treat the listed focus, execution bias, prompt style, batch bias, and avoid-lines as prompt guardrails.

Then choose freshness levers that can make the prompts feel alive without pushing them into the wrong lane.

Good places to vary:

- visual metaphor wording
- prompt density inside the allowed lane
- repeated motif language
- graphic system phrasing
- reveal language
- how continuity anchors are emphasized

Do not use freshness to smuggle in the wrong film grammar.

## Prompt Freshness Levers By Type Family

### Information-Led Types

Good freshness levers:

- graphic metaphor vocabulary
- card or module naming logic
- compare-state phrasing
- label hierarchy emphasis
- visual system keywords

Keep locked:

- clarity-first phrasing
- high graphic control
- low reliance on actor performance
- low cinematic clutter

### People-Led Types

Good freshness levers:

- support-evidence wording
- recurring place or object anchor
- emotion phrasing intensity
- cutaway family wording
- credibility detail emphasis

Keep locked:

- continuity-forward prompt order
- restrained motion language
- stable framing cues
- message readability

### Hype-Led Types

Good freshness levers:

- rhythm phrase system
- impact verbs
- reveal wording
- motif repetition
- title-hit cadence

Keep locked:

- short prompt units
- readable action
- quick retry friendliness
- low explanatory density

### Trust-Led Types

Good freshness levers:

- proof-detail emphasis
- environment character
- symbolic support motif
- transformation wording
- credibility cue selection

Keep locked:

- clean realism
- believable environments
- controlled language
- no hype-style excess

### Narrative-Led Types

Good freshness levers:

- motif lexicon
- transformation verbs
- emotional texture words
- reveal sequencing
- recurring symbolic anchors

Keep locked:

- coherent progression
- manageable motion arcs
- visual cause-and-effect
- no generic cinematic filler

## Product Promo Short

- Prompt focus: benefit clarity and visual payoff
- Best execution bias: `ai-first-still-to-video` plus selected `direct-video` shots
- Prompt style: product first, result visible, action easy to read
- Batch bias: group by same product and same use environment
- Avoid: concept-heavy style wording that hides the product

## Enterprise Promo

- Prompt focus: trust, capability, scale
- Best execution bias: mixed still-to-video and hybrid-reference
- Prompt style: cleaner environment and human credibility cues
- Batch bias: environment and brand-tone consistency
- Avoid: hard-sell ad prompt language

## Brand Image Film

- Prompt focus: brand feeling and signature mood
- Best execution bias: still-to-video with hero-frame control
- Prompt style: stronger motif language, lower literal feature density
- Batch bias: color and motif continuity
- Avoid: over-explaining operational detail in every shot

## Internal Promo / Internal Adoption Short

- Prompt focus: practical understanding
- Best execution bias: still-to-video, UI-led shots, graphic-composite assists
- Prompt style: direct, low-jargon, familiar workplace context
- Batch bias: same UI state, same task type, same office environment
- Avoid: over-dramatic cinematic language

## Opening Video

- Prompt focus: impact and reveal timing
- Best execution bias: hero stills, selected direct-video reveal shots, graphic-composite typography
- Prompt style: shorter, more forceful, more shape and atmosphere driven
- Batch bias: title cards, reveal beats, hero motifs
- Avoid: too many instructional details

## Warm-Up / Hype Video

- Prompt focus: momentum and energy
- Best execution bias: direct-video where motion matters, graphic-composite for text hits
- Prompt style: short impact phrases, readable action, strong transitions
- Batch bias: rhythm families and repeated impact motifs
- Avoid: long descriptive prompts that slow iteration

## Event Teaser

- Prompt focus: anticipation
- Best execution bias: reveal-oriented still-to-video plus graphic-composite
- Prompt style: fragmentary but controlled
- Batch bias: countdown assets, slogan assets, teaser motifs
- Avoid: fully resolving every visual mystery

## Event Recap

- Prompt focus: peak memory compression
- Best execution bias: direct-video for energy, still-to-video for select hero frames
- Prompt style: event moments, reactions, summary beats
- Batch bias: stage shots, audience shots, signage shots
- Avoid: trying to recreate every moment from the event

## Same-Day Edit / Flash Recap

- Prompt focus: speed and usable volume
- Best execution bias: short direct-video prompts, lightweight composite support
- Prompt style: simple, repeatable, fast to retry
- Batch bias: one event look, one contrast pattern, one typography layer
- Avoid: fragile continuity-heavy prompts

## MC / Guest Intro Video

- Prompt focus: identity and entrance framing
- Best execution bias: hero stills, graphic-composite name/title treatment
- Prompt style: compact, prestige-driven
- Batch bias: same intro layout system
- Avoid: building too many unique visual systems for one intro

## Program Interstitial / Segment Bumper

- Prompt focus: transition clarity
- Best execution bias: `graphic-composite`
- Prompt style: minimal and modular
- Batch bias: one bumper system, different titles
- Avoid: overproducing bumper shots like mini-films

## Interview Film

- Prompt focus: speaker continuity and quote support
- Best execution bias: hybrid-reference and still-to-video
- Prompt style: stable framing, stable light, controlled emotion
- Batch bias: one interview look, one cutaway family
- Avoid: excessive camera motion or over-stylized prompts

## Customer Testimonial

- Prompt focus: proof and sincerity
- Best execution bias: hybrid-reference or still-to-video
- Prompt style: concrete use context, real-looking work environment, low hype
- Batch bias: customer environment and proof shots
- Avoid: generic praise-language with no visual evidence

## Case Story Short

- Prompt focus: before/after legibility
- Best execution bias: still-to-video and compare-oriented graphic-composite
- Prompt style: contrast early, result clearly
- Batch bias: before-state family, after-state family
- Avoid: abstract "success" visuals

## Street Interview / Vox Pop

- Prompt focus: quick response variety
- Best execution bias: direct-video or reference-assisted capture style
- Prompt style: spontaneous but readable
- Batch bias: shared street backdrop and question card system
- Avoid: turning it into a polished corporate interview look

## Training / Process Explainer Short

- Prompt focus: step clarity
- Best execution bias: `graphic-composite`, UI demo, still-to-video support
- Prompt style: explicit process state, low ambiguity
- Batch bias: step templates and repeated process cards
- Avoid: metaphor-heavy prompts that obscure the workflow

## Policy / Compliance Communication

- Prompt focus: rule clarity and behavioral guidance
- Best execution bias: graphic-composite plus scenario shots
- Prompt style: direct, plain, consequence-aware
- Batch bias: scenario templates and warning card system
- Avoid: vague motivational phrasing

## Recruitment Promo

- Prompt focus: people, team energy, role appeal
- Best execution bias: still-to-video and hybrid-reference
- Prompt style: human, environment-aware, mission-aware
- Batch bias: team shots, culture shots, work-context shots
- Avoid: empty "great place to work" language with no supporting visuals

## Leadership Message Film

- Prompt focus: authority and message clarity
- Best execution bias: stable still-to-video and hybrid-reference
- Prompt style: restrained, credible, centered on the speaker
- Batch bias: one primary address look and support cutaways
- Avoid: flashy motion that distracts from the message

## MG Explainer / Information Short

- Prompt focus: layout clarity
- Best execution bias: `graphic-composite`
- Prompt style: icons, UI cards, labels, text hierarchy, compare states
- Batch bias: design system first, shot variations second
- Avoid: writing prompts like live-action scenes

## Data Visualization Film

- Prompt focus: metric readability
- Best execution bias: `graphic-composite`
- Prompt style: counters, charts, labels, guided emphasis
- Batch bias: reusable chart modules
- Avoid: overstuffed single-frame statistics

## UI Demo / Screenflow Video

- Prompt focus: screen understanding
- Best execution bias: `graphic-composite` and hybrid-reference
- Prompt style: interface state first, user action second, result third
- Batch bias: screen family, zoom family, annotation family
- Avoid: talking around the UI without locking the UI state

## Flash / Quick-Cut Montage

- Prompt focus: speed and emphasis
- Best execution bias: short direct-video and graphic-composite text hits
- Prompt style: shorter clauses, fewer moving parts per shot
- Batch bias: repeatable rhythm templates
- Avoid: long multi-action prompts

## Concept Narrative Short

- Prompt focus: motif and transformation
- Best execution bias: still-to-video with strong hero keyframes
- Prompt style: visual motif early, transformation clearly staged
- Batch bias: motif continuity and environment continuity
- Avoid: generic "cinematic" wording with no symbolic anchor

## Emotional Atmosphere Short

- Prompt focus: mood progression
- Best execution bias: still-to-video
- Prompt style: softer motion, sensory detail, lower prompt density
- Batch bias: light direction, palette, emotional motifs
- Avoid: over-explaining the feeling

## Trailer / Teaser Style Short

- Prompt focus: anticipation and withheld reveal
- Best execution bias: direct-video for dramatic beats, graphic-composite for title reveals
- Prompt style: sharper, more fragmentary, high contrast
- Batch bias: reveal families and title cards
- Avoid: over-describing the full narrative

## Documentary-Style Enterprise Film

- Prompt focus: authenticity
- Best execution bias: hybrid-reference and stable still-to-video
- Prompt style: observational, real-world, low hype
- Batch bias: real-space continuity and process continuity
- Avoid: glossy ad language that breaks documentary tone

## Type Selection Rules

- Follow the normalized `video_type` from Step 1 and the storyboard pattern from Step 2.
- If a format is information-led, increase `graphic-composite` usage and reduce cinematic wording.
- If a format is people-led, simplify motion language and increase continuity locks.
- If a format is hype-led, shorten prompts and prioritize readable impact over nuance.
