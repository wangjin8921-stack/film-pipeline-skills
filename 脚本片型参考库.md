# 脚本片型参考库

基于 `E:\视频素材\做片看\脚本类` 目录下全量 `ppt / pptx` 的第一版整理。

当前已扫描文件数：

- `59` 个

这份库的目标不是做“客户名单”，而是做“可复用片型库”。  
你后面写 `/film` 时，优先先从这里选片型，再换成具体客户、主题、风格。

## 使用规则

先判断你的项目更像哪一类：

- 大会 / 峰会 / 发布会 / 年会 / 医学论坛开场
- 企业宣传片 / 行业方案片
- 品牌形象片 / 品牌里程碑片
- 医疗患教 / 流程讲解 / 项目说明
- 颁奖 / 串场 / 嘉宾引入
- AI / 数智 / 科技概念片
- 团队片 / 招聘片 / 宣言片

然后再选推荐的 `video_type`。

## 片型 1：大会 / 峰会 / 发布会 / 年会 / 医学论坛开场

推荐 `video_type`：

- `Opening Video`

这类片子的共性：

- 用一个大主题或大问题开场
- 快速建立气氛、价值感、时代感
- 往往分 3 段左右展开
- 最后落到现场 handoff、主题揭幕或嘉宾上场

适合的表达：

- slogan、大字字幕、章节词
- 光线、粒子、科技城市、历史文化、行业符号
- 由情绪或气势带入会议主题

## 给 `/film` 的快速决策

如果你以后懒得想，先按这个逻辑选：

- 活动启幕、论坛、峰会、发布会：`Opening Video`
- 企业能力、行业方案、工厂、平台：`Enterprise Promo`
- 品牌情绪、品牌故事、里程碑：`Brand Image Film`
- 医疗项目说明、流程、患教：`Training / Process Explainer Short`
- 颁奖、嘉宾、舞台节点：`MC / Guest Intro Video`
- AI、未来感、数字人概念：`Concept Narrative Short`
- 团队士气、内部宣言、启动会：`Internal Promo / Internal Adoption Short`
